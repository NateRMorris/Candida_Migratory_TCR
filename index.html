R Scripts for scRNA/Cite-Seq Analysis

Scripts related to scRNA analysis and clustering.

  • Candida_sc_QC.Rmd - Quality control for single-cell RNA-seq data.
  • Candida_sc_Analysis_11_02_2023.Rmd - Analysis of single-cell data, including dimension reduction, clustering, differential abundance analysis, identifying marker genes, cell type annotations, pseudotime analysis and cell-signaling analysis (date: 11/02/2023).
  • Candida_sc_Analysis_11_02_2023.Rmd - Analysis of single-cell data updated version (date: 01/08/2024).
  • Candida_sc_clust5_analysis.Rmd - Sub-clustering analysis for particular cluster of interest in scRNA-seq data.
    • Candida_sc_Analysis_WNN.Rmd - Weighted nearest neighbor analysis for integration of scRNA with Cite-Seq surface protein assay.

GitHub Repository: R Scripts Repository

R Scripts for TCR Analysis and Integration

Scripts related to TCR analysis and diversity.

  • Candida_tcr_paired.Rmd - Indentifying and analyzing TCR sequences that are identical and found in cells from multiple organs, also considered "paired across organ", or migratory between organ.
  • Candida_tcr_duplicate.Rmd - Identification and analyzing "duplicate" TCR sequences, which are found in multiple cells and therefore are proliferating.
  • Candida_tcr_seurat_integration.Rmd - Integration of TCR data with Seurat scRNA data.
  • Candida_tcr_Diversity.Rmd - Diversity analysis of TCR sequences.
  • Candida_tcr_clust5.Rmd - TCR analysis of particular cluster of interest.
  • Candida_tcr_Gearty_assumption.Rmd - Analysis based on Gearty's assumptions.
  • Candida_tcr_Cytoscape.Rmd - Visualization using Cytoscape.
